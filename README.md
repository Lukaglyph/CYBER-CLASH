🚀 CYBER CLASH
ネオン煌めくサイバー空間を舞台にした、ブラウザベースのローグライク・全方位シューティングゲームです。迫りくる敵の弾幕を相殺し、経験値を集めて機体を究極まで進化させましょう。

✨ 主な機能
弾消しシステム: 自機の弾で敵の弾を相殺可能。防御が最大の攻撃になります。

ビルド構築: レベルアップ時にランダムに提示される3つのアップグレードから1つを選択。

強力なスキル:

🟢 拡散レーザー: 基本となる連射武器。

🟡 追尾ミサイル: 敵を自動で追跡し、高火力を叩き込む。

🔵 衝撃波シールド: 周囲の弾を一掃する緊急回避手段。

モバイル対応: タッチ操作、マウス操作の両方に最適化。

🎮 遊び方
操作方法
PC: マウスでドラッグして機体を移動。

Mobile: 画面をスワイプして機体を移動。

ゲームのルール
生き残れ: 時間の経過とともに敵の攻撃は激しくなります。

弾を消せ: 敵の弾に自分の弾を当てると、弾が消えて**経験値（XP）**が出現します。

進化しろ: XPを集めてランクアップし、武器を強化してください。

🛠 技術スタック
Language: HTML5 / CSS3 / JavaScript (ES6+)

Rendering: Canvas API (No external libraries used)

UI Design: CSS Neon Glow Effects & Flexbox
